# Bob's Burgers Bio Expander
[View Demo](https://codepen.io/karlyhoffman/full/NyPbLv/)

## Project Deliverables:
- A photo gallery that displays bios when a user clicks on a photo.

## Technologies Used:
- HTML 5
- CSS/SCSS
- JavaScript/JQuery
- Gulp

## Getting Started:
1. Download or clone repository
2. Run `npm install`
3. Run `gulp`
